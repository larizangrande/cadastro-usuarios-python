# 👩🏻‍💻 Sistema de Cadastro de Usuários

Projeto desenvolvido em Python para praticar lógica de programação, listas, condicionais e estruturas de repetição.

## 🚀 Funcionalidades

- Cadastro de usuários
- Listagem de usuários
- Busca de usuários
- Exclusão de usuários
- Menu interativo

## 🛠 Tecnologias utilizadas

- Python 3

## 🎯 Objetivo

Praticar conceitos fundamentais da linguagem Python e desenvolver um sistema simples de gerenciamento de usuários.

## 📸 Exemplo

```bash
==============================
 SISTEMA DE CADASTRO
==============================
1 - Cadastrar usuário
2 - Listar usuários
3 - Buscar usuário
4 - Excluir usuário
5 - Sair
```
